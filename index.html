markdown上手

以前自己也记录过很多的笔记，内容和排版都是随意书写，我现在都不知道以前写的东西有什么用 (╥╯^╰╥)。
只有看了下 stormzhang 的博客也有一种想自己写写的想法。我喜欢简单又快速的书写。刚好markdown又能装 B还好用，欣然的喜欢上了。

使用 markdown 的优点
  • 专注你的文字内容而不是排版样式。你可以飕飕的一直往下写。
  • 轻松的导出 HTML、PDF 和本身的 .md 文件。
  • 纯文本内容，兼容所有的文本编辑器与字处理软件。
  • 可读，直观。适合所有人的写作语言。
markdown 编辑器推荐

  • Windows
    马克飞象：你可以下载客户端或者是直接在浏览器使用（果冻我正在使用）。支持在线预览，并且写好的文章可以直接同步到印象笔记。
    Cmd Markdown：支持在线预览。可以参考这篇知乎专栏
    小书匠：功能很强大，问题是部分功能收费… 估计是插件太多，页面有点卡顿。

  • Mac
    Mou ：页面简单，支持预览。
    Ulysses Ⅲ： 对于编辑爱好者可以看看这个

如果你还在纠结用什么 IDE，那选择 markdown干嘛，直接在 wrold 里编辑多好。他的语法不多又很好掌握。多用用，就会了。

markdown 语法介绍

就简要的介绍下，其实在推荐的 IDE 里有的包含 markdown 语法的说明（马克飞象）。

标题

这个简单吧，想标题字号小，你就使劲加 # ，如果加上没有效果，看看是不是 # 号后面没有空格，或者不是在起实行。

列表

引用

图片链接

其实就比插入链接的语法前多了个！号；
添加图片描述：组成图片链接的 [ ] 里的文字前面加上@ 「@插入图片的例子」就会在图片的下面显示对图片描述内容。

粗体和斜体
  • 用两个 包含文字就是粗体：「你**」
  • 用一个 包含文字就是斜体：「你*」
